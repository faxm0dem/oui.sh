# oui.sh

Le CLI essentiel pour jouer les citations de http://ouich.es

## Utilisation

À mettre dans votre `~/.bashrc` ou équivalent :

```
. oui.sh
```

Citation au hasard :

```
ouich
```

Citation choisie :

```
ouich julienlepers
```

Liste des citations :

```
ouich list
```

## Dépendances

Installation (debian, ubuntu) :

```
sudo apt install mpv alsa-utils lftp bsdextrautils
```
